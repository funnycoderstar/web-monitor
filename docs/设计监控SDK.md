## SDK 的思路

区分业务，性能，稳定性埋点

一个好的技术方案文档

-   要解决的问题
-   有哪些方案
-   方案详细对比，优缺点，
-   得出分别使用的场景

检测到异常后，如何快速排查可能的问题，以及如何验证异常已经修复

容易混淆的概念区别
白屏异常，页面崩溃，卡顿
